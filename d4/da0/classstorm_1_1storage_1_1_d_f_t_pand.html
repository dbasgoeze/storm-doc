Detailed Description

template<typename ValueType>
class storm::storage::DFTPand< ValueType >

Priority AND (PAND) gate.

Fails if all children fail in order from first to last child. If a child fails before its left sibling, the PAND becomes failsafe. For inclusive PAND<= gates, simultaneous failures are allowed. For exclusive PAND< gates, simultaneous failure make the gate failsafe.
