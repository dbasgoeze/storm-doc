Detailed Description

template<typename ValueType, bool Nondeterministic>
class storm::modelchecker::helper::SparseInfiniteHorizonHelper< ValueType, Nondeterministic >

Helper class for model checking queries that depend on the long run behavior of the (nondeterministic) system.

Template Parameters
ValueTypethe type a value can have
Nondeterministictrue if there is nondeterminism in the Model (MDP or MA)
