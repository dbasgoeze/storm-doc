Detailed Description

This class implements the visitor part of the visitor pattern.

Every statement accepts such a visitor which then again calls the corresponding visit method of the visitor object. In such a way double dynamic dispatching can be realized. Every structure or function that requires to differentiate between concrete statement instantiations (such as if statements, assignments statements, ...) should be handled by a concrete visitor implementation.
