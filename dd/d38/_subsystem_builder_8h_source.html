36  /*
37  * Removes all states and actions that are not part of the subsystem.
38  * A state is part of the subsystem iff
39  * * it is selected in subsystemStates AND
40  * * keepUnreachableStates is true or the state is reachable from the initial states
41  * An action is part of the subsystem iff
42  * * it is selected in subsystemActions AND
43  * * it originates from a state that is part of the subsystem AND
44  * * it does not contain a transition leading to a state outside of the subsystem.
45  *
46  * If this introduces a deadlock state (i.e., a state without an action) it is either
47  * * fixed by inserting a selfloop (if fixDeadlocks is true) or
48  * * an exception is thrown (otherwise).
49  *
50  * @param originalModel The original model.
51  * @param subsystemStates The selected states.
52  * @param subsystemActions The selected actions
53  * @param keepUnreachableStates if true, states that are not reachable from the initial state are kept
54  */
