Detailed Description

template<typename ValueType>
class storm::storage::geometry::PolytopeTree< ValueType >

Represents a set of points in Euclidean space.

The set is defined as the union of the polytopes at the leafs of the tree. The polytope at inner nodes should always be the convex union of its children. The sets described by the children of a node are disjoint. A child is always non-empty, i.e., isEmpty() should only hold for the root node.
