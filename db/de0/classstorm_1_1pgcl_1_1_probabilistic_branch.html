Detailed Description

This class represents a probabilistic branch.

It branches into two subprograms. The first program is executed with the probability p given by the assigned expression, the second subprogram is executed with probability (p - 1).

Constructs a probabilistic branch initialized with the given probability and the left (first) and right (second) subprograms.

Note that no verification is made whether the probability lies between 0 and 1 is made here. This is done at runtime of the PGCL program.
